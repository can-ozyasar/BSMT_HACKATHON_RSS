# BIOS Signal Radar — Frontend (React + Vite)

UI, backend ile `http://localhost:8000` üzerinden konuşacak şekilde tasarlanmıştır.

## Çalıştırma

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notlar

- Backend kurulum ve env ayarları için `../backend/.env.example` ve repo kökündeki `README.md` dosyasına bakın.
